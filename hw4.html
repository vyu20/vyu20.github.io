List Usage
Unordered List An unordered list is used to present a list of items that do not have to be shown in any specific order and to start this list you use the tag <ul>.
Ordered List An ordered list is used to present a list of items that need to be shown in a specific order and to start this list you use the tag <ol>.
Definition List A definition list is used when items need to come in pairs such as for glossary and vocabulary and to start this list you use the tag <dl>.
